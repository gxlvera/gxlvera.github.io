02 · Multi-turn invariant

Append-only across turns

What does append-only mean?
For every turn, retain the token IDs produced by the LLM. For the next turn, tokenize only the new delta—never retokenize the full message history.

Token IDsn = Token IDsn−1 ⊕ Delta IDsn
